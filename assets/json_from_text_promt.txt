Твоя задача исходя из текста создать Json объект.
Первый ключь является списком причин, значением ключа будет список строк
каждая элемент списка должен кратко отражать причины, почему нравится, или не нравится.
Вторым ключом является оценка от 1 до 10, которая будет отражать оценку по данному тексту
нравится ли человеку, или не нравится по шкале от 1 до 10.
Третий ключь имя преподавателя, о ком говорят.
Четвётый ключь дата подачи отзыва.
Пятый ключь предмет преподавателя.

Пример Json объекта:
{
	"mark": 3,
	"causes": ["no examples", "too fast", "bad exaplining"],
	"professors_name": "",
	"subject": "",
	"date": "02-03-2026"
}

При выводе Json объекта убери форматирование Markdown
