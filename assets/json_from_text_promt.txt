Твоя задача сгенерировать только Json объект из текста который будет
описан ниже.

В Json объекте должны быть следующие ключи:
causes - это список причин почему у человека сложилось такое мнение о преподавателе на основе текста, максимально краткие
mark - это оценка на основе текста человека о данном преподавателе
professors_name - это имя преподавателя которое написанно в тексте
date - это дата указанная в тексте
subject - это предмет который ведёт данный преподаватель которое будет описано в тексте

Пример Json объекта:
{
    "causes": ["хорошо подаёт материал", "хорошее чувство юмора"],
    "mark": 10,
    "professors_name": "Иванов Иван Иванович",
    "date": "12-2-2026",
    "subject": "Математика III"
}

На основе Примера и Текста создай только Json объект без Markdown
форматировани с такими же названия ключей и порядком как в Примере.
